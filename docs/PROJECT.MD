# DevDeploy

## Vision

DevDeploy is a production-grade deployment platform for containerized Django applications.

The goal is to automate infrastructure provisioning and application deployment using Infrastructure as Code and CI/CD.

This project is inspired by platforms such as Render, Railway and Heroku.

The Django Financial Authentication Service is only the reference application.

The real product is the deployment platform.
